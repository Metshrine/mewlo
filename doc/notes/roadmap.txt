Roadmap
=======

1. Database layer
	a) Persistent settings in database
	b) Log to database


2. Plugin package database updating


3. View/template system


4. Menu system
	a) NavNodes
	b) View support of top and sidebar menus


5. Users
	a) registration, login, logout
	b) ACL system
	c) profile (avatars, arbitrary properties)
	d) bridged login
	e) messaging system


6. Verification system


7. Groups
	a) creating
	b) applying
	c) inviting
	d) managing


8. File handling system
	a) uploads
	b) downloads
	c) admin management
	d) statistics


9. Table/Crud support system
	a) browsing
	b) searching
	c) sorting
	d) filtering
	e) batch operations
